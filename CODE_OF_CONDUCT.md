# Code of Conduct

## Our Commitment
Escape Launcher is dedicated to providing a welcoming and inclusive environment for all contributors. We do **not tolerate discrimination, harassment, or abusive behavior** of any kind based on age, race, gender, sexual orientation, disability, religion, nationality, or any other characteristic.

## Expected Behavior
- Be respectful and professional in all interactions.
- Use inclusive language (Use "they" for users whose pronouns you don't know). 
- Treat others’ opinions and contributions with courtesy.
- Do not be rude about the quality of anyones code, remember everyone was new to development at some point and constructive criticism will be much more valuable than pure hate.

## Where This Applies
This Code of Conduct applies to **all interactions in this repository**, including but not limited to issues, pull requests and comments.

## Reporting
If you experience or witness behavior that violates this Code of Conduct, you can:
- Block the user if necessary.
- Report the behavior by tagging the repo owner (me, GeorgeClensy) on the relevant issue, pull request, or comment.
- Optionally contact the me via Discord on our server or a DM.
